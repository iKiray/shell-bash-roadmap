# Vim Practice

This guide provides sequences to practice Vim commands on `practice1.txt` and `practice2.txt`.

## Practice Sequence for practice1.txt

1. Press `i` to enter Insert mode.
2. Type a few sentences.
3. Press `Esc` to return to Normal mode.
4. Use `w` and `b` to move by words.
5. Press `dd` to delete a line.
6. Press `u` to undo.
7. Press `yy` then `p` to duplicate a line.
8. Search using `/word`.
9. Save with `:w`.
10. Exit with `:q`.

## Practice Sequence for practice2.txt

Then work through this sequence:

1. `G` → jump to the bottom.
2. `gg` → return to the top.
3. `j` twice → move to server3 online.
4. `yy` → copy the line.
5. `p` → paste a duplicate below it.
6. `/offline` → search for the first offline.
7. `n` → jump to the next offline.
8. `0` → move to the beginning of the line.
9. `w` twice → move to the status word (offline).
10. `dw` → delete offline.
11. `i` → enter Insert Mode and type online.
12. `Esc` → return to Normal Mode.
13. `u` → undo the change.
14. `Ctrl-r` → redo the change.
15. `:w` → save the file.
16. `:q` → quit.

## Notes

- `practice1.txt` is a ChatGPT-generated sequence practice file.
- `practice2.txt` is a ChatGPT-generated sequence practice file that contains a server status list for practicing search, navigation, and editing.
