# Vim Practice

This guide provides a sequence to practice Vim commands on `practice.txt`.

## Practice Sequence

1. Press `i` to enter Insert mode.
2. Type a few sentences.
3. Press `Esc` to return to Normal mode.
4. Use `w` and `b` to move by words.
5. Press `dd` to delete a line.
6. Press `u` to undo.
7. Press `yy` then `p` to duplicate a line.
8. Search using `/word`.
9. Save with `:w`.
10. Exit with `:q`.

## Notes

- `practice.txt` is a ChatGPT-generated sequence practice file.
